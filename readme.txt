
TexStudio 编译生成 pdf 时 需注意 


中文需   Options --->  Configure TexLaTex ----> Build  ----> Default Compiler ---> XeLaTex

英文需  Options --->  Configure TexLaTex ----> Build  ----> Default Compiler ---> XeLaTex 或 PdfLaTex 或 LaTex 皆可
